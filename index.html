About

OoT Online is a free to use, open source modification of Ocarina of Time that allows people to play together.


OoT Online comes bundled with the ModLoader as the first plugin created for it. This plugin gives you the ability to share your OoT experience with friends. See your friends in the game world, share items and quest progress with them, and push them off ledges as you adventure through this timeless classic together.


Setup


How to run OoT Online as a client:

  • Step 1: Click Download Now! and download the newest client version.
  • Step 2: Extract the zip into an empty folder.
  • Step 3: Put your 1.0 NTSC (US/JP) OoT ROM in the "rom" folder.
  • Step 4: Open "OoTModLoader.exe"
  • Step 5: If "Windows Security Alert" shows up, make sure the private network box is checked and click "allow access".
  • Step 6: During the first-time setup, the program will install bizhawk. Wait for the button to finish installing bizhawk.
  • Step 7: After Bizhawk is installed, close the program and reopen it.
  • Step 8: You should see the config loaded. Change the settings accordingly to your preferences. Make sure you have the correct server and port information typed in mentioned in Known Issues & FAQ.
  • Step 9: Once you are ready to play, click "Connect to Server".
  • Step 10: If all is setup properly, Bizhawk should load the emulator with the ROM.
  • Step 11: Configure the emulator settings to your preferences and you should be ready to go!

Known Issues & FAQ

Inverted Controls

Your Left/Right or Up/Down is inverted. In bizhawk, click "Config -> Controls" and go to the third tab called "Analog Controls". If the L/R is inverted, make sure sensitivity on the top is -100 (Or if its already -100, to 100) , otherwise if the U/D is inverted, same instructions just for the bottom sensitivity option.

Low FPS

This is normal for bad computers. The best we can suggest is using Rice video plugin in N64 -> Plugins as well as keep the video resolution listed at the bottom of the plugins page to something lower such as 400x300.

Glitchy Graphics

If you are using a plugin other than GlideN64, this is normal and a flaw of the emulator plugins, not the ModLoader, so we cannot fix this. You can try another graphics plugin or if you are brave, tweak some settings in the graphics plugin.

How do I host a server?

This is usually asked for the purpose of playing with the friends. In the case of OoT Online, you are not required to host a server to play a private game with your friends. Simply do the following:
  • Make sure the Server IP and Port are set to 192.99.70.23 and 8082.
  • Make sure to put any lobby name and password that you want. This creates a session for your group to play in.
  • Give that SAME lobby name and password (case sensitive in both text boxes) to your friends.
  • Play the game.

Long Delay on Start Menu


Pressing start to access the pause menu may take a long time. In bizhawk, click "Tools -> Cheat Code Converter" and enter in "Pause Menu Fix" for description and "801DA5CB 0002" in the code box and click convert.

Hosting a Server

For confusion sake, hosting a server is not made as easily accessible. If you wish to play in a private session with your friends, simply set a custom lobby name and password and start the game. Give the same lobby name and password (case sensitive) to your friends for them to join. If you are familiar with port forwarding and server setup and wish to host for a lan or more performance, open the OotModLoader-Config.json file in the Oot Online folder and set "isMaster" to true. This should only be done on the person hosting a server, not on the other players wishing to connect, they only need the Server IP and Port you are hosting on.

I can't see my friends in game

This can be caused for a few reasons, here is a check list to help identify and/or fix the problem:
  • Allow the game through firewall and antivirus
  • Make sure you put in the exact same lobby name and password (it is case sensitive)
  • Make sure the Server IP and Port are set to the master server, you can press reset to get the default one.
  • Open OotModLoader-config.json with notepad and make sure "isMaster" is set to false
  • If the steps above dont work, completely disable antivirus. (This is mainly targetted at AVG and Avast as known culprits. We reccommend Bitdefender, it's a better antivirus)

Black Screen Glitch

If you are able to hear the audio but the screen is black, this is because the current graphic plugin does not work on your pc for what ever reason. Click the N64 tab on the emulator and try using the other plugins to find one that works (rice is suggested). Note: Other plugins might have small graphical glitches, but it's better than a black screen.

Some items (like keys) aren't synced

This is normal. Keys, Rupees, generally anything with a counter indicating an 'amount' do not sync. All other items and flags however should properly sync. If someone happens to take a key and leave the game, the codes to get them can be found here

My game is pixelated

This fix might reduce FPS, but when the game is running, open the N64 tab at the top, then Plugins. Then click the bottom dropdown menu for Video Resolution and set your resolution to something higher.

The other players have bugged animations

Disable the lua script by double clicking it, exit the room, restart the lua script, then enter the room again.

Getting killed by Phantom Ganon causes the elevator to get stuck

Use the following gameshark code to teleport back to Phantom Ganon's room until this issue can be resolved in a future version.
816000AC 000C
816000AE 0000
806000AB 0005
