OoT - Online

Play & Run Together :)

About

OoT Online is a free to use, open source modification for Ocarina of Time to play together.


OotOnline comes bundled with the ModLoader as the first plugin created for it. This plugin gives you the ability to share your Ocarina of Time experience with friends. See your friends in the game world, share items and quest progress with them, and push them off ledges as you progress through this timeless classic together.


Setup


How to run the OoT Online as a client

  • Step 1: Go to github and download the latest GUI version zip.
  • Step 2: Extract the zip into an empty folder.
  • Step 3: Put your 1.0 NTSC (US/JP) OoT ROM in the "ROM" folder.
  • Step 4: Open "OoTModLoader-GUI.exe"
  • Step 5: If "Windows Security Alert" shows up, make sure the private network box is checked and click "allow access".
  • Step 6: During the first-time setup, the program will install bizhawk. Wait for the button to finish installing bizhawk.
  • Step 7: After Bizhawk is installed, close the program and reopen it.
  • Step 8: You should see the config loaded. Change the settings accordingly to your preferences. Make sure the "Server" tab has the right information in it, like the server IP.
  • Step 9: Once ready, press "Connect to Server".
  • Step 10: If all is setup properly, Bizhawk should load the emulator with the ROM.
  • Step 11: Configure emulator settings to your preferences. Afterwards you are ready to go!
